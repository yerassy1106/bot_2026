## 📌 Project Overview
**CFO Bot** is an intelligent calculator agent designed to estimate monthly cloud infrastructure costs across major providers (AWS, Azure, GCP). This project was developed for the "Senior Information Systems" course, utilizing the **Spec-Driven Development (SDD)** methodology via Google Antigravity.

## 🎯 Project Objective
To automate financial planning for cloud resources using rigorous mathematical modeling and a **Single Source of Truth (SSOT)** to ensure billing transparency and accuracy.

---

## 📂 Project Deliverables (Phase 1-4)

In accordance with project requirements, the documentation is structured into the following artifacts:

1.  **[Phase 1: SSOT (System Specification)](./SSOT.md)**  
    *The Single Source of Truth. Defines components, unit prices, and mathematical formulas.*

2.  **[Phase 2: Implementation Plan](./Implementation_Plan.md)**  
    *The technical roadmap generated by AI agents based on the SSOT.*

3.  **[Phase 2: Test Specifications](./Test_Specification.md)**  
    *A suite of verification tests to ensure 100% calculation accuracy for Compute, Storage, and Database tiers.*

4.  **[Phase 4: Pricing Strategy](./Pricing_Strategy_Document.md)**  
    *Business strategy and justification for the selected cloud architecture based on bot outputs.*

## 🛠 Tech Stack (Architecture)
*   **Frontend:** React.js + Vite (v8)
*   **Styling:** Tailwind CSS (v4)
*   **Deployment:** Google Firebase Hosting
*   **Methodology:** Spec-Driven Development (SDD)

## 📊 Mathematical Logic (SSOT Model)
The bot calculates costs using the following formula:
$$Total = (Hours \times C_{unit}) + (Storage \times S_{unit}) + (Bandwidth \times B_{unit}) + DB_{tier}$$
